# USS-Assets
Python application to track USS aircraft carrier locations using publicly available information.

## ⚠️ Important Disclaimer
This application only uses **publicly available information** from official sources like:
- US Navy public affairs releases
- Defense news publications  
- Maritime tracking services (historical data)

Real-time military vessel positions are classified for security reasons.

## Features
- Track USS aircraft carrier deployments
- Monitor Middle East region operations
- Scrape official Navy news sources
- Generate intelligence reports from public data
- **Interactive visual maps with direction arrows**
- **MarineTraffic API integration for vessel positions**
- **Real-time course and speed tracking**

## Usage

### Basic Tracker
```bash
python uss_tracker.py
```

### Visual Map Tracker (with MarineTraffic API)
```bash
python3 visual_tracker.py
```

## Installation
```bash
# Install dependencies
python3 -m pip install -r requirements.txt
```

## Output
- Console reports with current deployments
- JSON files with detailed intelligence data
- **Interactive HTML maps (uss_carrier_map.html)**
- Focus on Middle East region operations
- **Visual direction arrows and course tracking**
